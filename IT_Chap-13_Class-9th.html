III. Short Answer Questions

  1. What is Rapid Typing Tutor?
    Answer: Rapid Typing tutor is a free and open source software (FOSS) designed to learn typing skills on the computer. It can be used by both beginners and advanced learners.
  2. Main components of Rapid Typing Tutor window:
    Answer:
    • Title Bar – Shows software name and mode tabs (Current Lesson, Statistics, Editor).
    • Options & Control Buttons – Minimize, Maximize, Close, and Settings.
    • Toolbar – Allows lesson selection and control.
    • Text Panel – Displays the typing text.
    • Status Bar – Shows tips, progress, and volume.
    • Virtual Keyboard – Assists in touch typing.
  3. Why looking at the keyboard is not allowed in touch typing?
    Answer: It reduces speed, accuracy, and focus, breaks typing flow, and prevents muscle memory development. Touch typing relies on practicing key positions by feel.
  4. How to download Rapid Typing Tutor?
    Answer: Visit rapidtyping.com to download the latest version and access typing tools, games, and features.
  5. Base position for fingers:
    Answer: (a) Keyboard: Home Row – Left: A-S-D-F, Right: J-K-L-;, Thumbs on Spacebar.
    (b) Numeric Keypad: Index: 4, Middle: 5, Ring: 6, Thumb: 0.
  6. Use of Options button:
    Answer: Displays a list of options to control Rapid Typing settings. Minimize, Maximize, and Close buttons function normally.
  7. Lesson editor usage:
    Answer: Add, insert, change, delete lessons, group them in courses, create new courses, and review metrics.
  8. View results of a typist lesson:
    Answer: Select the Student Statistics tab in the main window to view results in graphical or table view.
  9. Units for measuring typing speed:
    Unit Description Formula
    WPM Words typed per minute. WPM = (Words without errors + Words with errors) / Minutes
    Net WPM WPM excluding errors. Net WPM = WPM - (Words with errors / Minutes)
    CPM Characters typed per minute. CPM = (Characters without errors + Characters with errors) / Minutes
    Net CPM CPM excluding errors. Net CPM = CPM - (Characters with errors / Minutes)
    KPM Keystrokes per minute. KPM = (Keystrokes without errors + Keystrokes with errors) / Minutes
    Net KPM Keystrokes per minute excluding errors. Net KPM = (Keystrokes without errors + Keystrokes with errors) / Minutes
  10. Net typing speed calculation:
    Answer: Net WPM = WPM - (Words with errors / Minutes)
  11. Typing accuracy:
    Answer: Percentage of correctly typed characters or words compared to total typed.
  12. Importance of maintaining rhythm:
    Answer: Keystrokes should come at equal intervals to control constant typing speed. Slowdown indicators help maintain rhythm.
