IV. Short Answer Questions

1. What is a file and a folder?
Answer-
A file is a collection of information stored on the computer buy a specific name at a specific memory location.
A Folder is a container you can use to store our organize files in a computer.


2. What happens when we power on the computer?
Answer-
When we power on the computer, it automatically runs a basic program called BIOS (Basic Input/Output System). The BIOS first does a self-test. If the self-test shows that the system is fine, the BIOS will load the Operating System. This means that the computer’s operating system is now ready to take inputs from the user.

3. What is the difference between copy and move operations?
Answer-
Copy operation creates a duplicate of a file or folder in another location, while move operation shifts the file or folder from one place to another.

4. Describe the functions of at least 5 types of keys.
Answer-
1.Navigation keys move the cursor up, down, right, and left within documents; HOME, END, PAGE UP, and PAGE DOWN help quickly move through text.
2.- The Windows key opens the Start Menu instantly, making navigation easier.
3.- Command keys include INSERT, DELETE, and BACKSPACE, used for overwriting and removing text or objects.
4.- Function keys (F1–F12) perform special functions that vary by software, such as help screens or shortcuts.
5.- Control keys (CTRL, SHIFT, ALT, SPACEBAR, CAPS LOCK, TAB) perform specific tasks depending on use.

5. Describe the functions of a mouse.
Answer-
Mouse is a pointing device. We can use to perform point, click double click, roll over/hover, and dragging action on the computer screen. .

6. How can you create the copy of a file on the desktop using mouse and the Ctrl key?
Answer-
Press and hold the Ctrl key, then drag the file using the mouse to another location. It will create a copy of the file.

7. What is a subfolder?
Answer-
A folder can also store other folder(s) inside it. A folder contained inside another folder is called a sub folder. You can create any number of sub folder in each can hold any number of files and additional sub folders.
