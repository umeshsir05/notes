IV. Short Answer Questions

1. What is a file and a folder?
Answer-
A file is a collection of information stored on the computer. A folder is a container used to store files and organize them properly.
2. What happens when we power on the computer?
Answer-
When we power on the computer, it performs a self-check, loads the operating system into memory, and gets ready for use.
3. What is the difference between copy and move operations?
Answer-
Copy operation creates a duplicate of a file or folder in another location, while move operation shifts the file or folder from one place to another.
4. Describe the functions of at least 5 types of keys.
Answer-
Alphabet keys: Used to type letters.
Number keys: Used to type numbers.
Function keys (F1–F12): Perform special tasks.
Arrow keys: Used to move the cursor in different directions.
Enter key: Used to start a new line or execute a command.
5. Describe the functions of a mouse.
Answer-
A mouse is a pointing device used to move the cursor, select items, open files or folders, drag and drop objects, and perform various operations through clicking.
6. How can you create the copy of a file on the desktop using mouse and the Ctrl key?
Answer-
Press and hold the Ctrl key, then drag the file using the mouse to another location. It will create a copy of the file.
7. What is a subfolder?
Answer-
A subfolder is a folder that is created inside another folder to organize files more efficiently.
