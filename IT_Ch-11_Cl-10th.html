III. Short Answer Questions.

1. What do you understand by the Frame style in LibreOffice Writer?
Answer-
In LibreOffice Writer, Frame Style refers to predefined formatting settings applied to frames—movable containers that hold text, images, or other objects. Frame styles control properties like size, position, borders, background, and text wrapping. They help maintain consistent design throughout the document. Users can apply or customize frame styles for tasks such as image captions or sidebars. This feature is especially useful for creating structured layouts in newsletters, brochures, or multi-column documents efficiently and uniformly.

2. What do you understand by Styles in context of the LibreOffice Writer word processor?
Answer-
In a word processor, a style is a set of text formatting characteristics, such as font size, color, and alignment which can be applied to a paragraph or any other selected piece of text in a document Styles help in giving a consistent look to a document and also make is easier to apply major formatting changes to a document. For example, you can make the font of all headers/footers in your document of a specific type just by clicking on a related style option.

3. Differentiate between Paragraph styles and Character Styles.
Answer-
Paragraph styles in LibreOffice Writer format entire paragraphs, controlling alignment, spacing, and indentation. Character styles apply only to selected text, affecting font, size, color, or emphasis like bold or italic. While paragraph styles define the document’s structure, character styles are used for highlighting specific words without altering the overall layout.

4. How will you filter the styles visible in the Styles deck?
Answer-
To filter styles in the Styles deck of LibreOffice Writer, open the Styles deck using F11. Use the drop-down menu at the top to choose options like All Styles, Applied Styles, or Custom Styles. This helps display only the relevant styles for easier formatting and document management.

5. Describe the Fill format mode in LibreOffice Writer.
Answer-
You can use the Fill format mode to apply a style to many areas quickly in your document, without having to go back to the Styles deck and double click the style every time. This method is very useful to format scattered text (paragraphs, words) and other elements with the same style. This also saves you from making multiple selections and then applying a same style to format them all.

6. How will you apply a Paragraph Style from the Styles deck to a paragraph?
Answer-
To apply a Paragraph Style from the Styles deck in LibreOffice Writer, place the cursor anywhere in the paragraph you want to format. Open the Styles deck by pressing F11, click on the Paragraph Styles icon, then double-click the desired style. The paragraph will immediately adopt the selected style.

7. What is the use of the Load Styles option?
Answer-
The Load Styles option in LibreOffice Writer allows you to import styles (like paragraph, character, frame, or page styles) from another document or template. It helps maintain consistency across documents by reusing formatting styles without recreating them manually, making document formatting faster, uniform, and more efficient.

8. How will you apply a Page style to a document?
Answer-
To apply a Page Style in LibreOffice Writer, follow these steps: 1. Go to the Styles deck (press F11). 2. Click the Page Styles icon. 3. Double-click the desired page style to apply it to the current page. 4. Alternatively, go to Format > Page Style and select a style.

9. Name the styles that can be modified and that cannot be modified using the Styles deck.
Answer-
In LibreOffice Writer, Paragraph, Character, Frame, Page, and List styles can be modified using the Styles deck. However, Table styles cannot be modified directly. Manual formatting and styles from read-only documents also can’t be changed through the Styles deck, making them limited in terms of customization and control.

10. What is the use of the New Style from Selection option in Style actions menu?
Answer-
The New Style from Selection option in LibreOffice Writer’s Style actions menu allows you to create a new style based on the formatting of selected text or a paragraph. It saves the current formatting (like font, size, color, spacing) as a reusable style, helping maintain consistency in the document.

11. What is text wrapping? Discuss any two text wrapping options that help you to position an image with the text in a document.
Answer-
Text wrapping is a feature that controls how text flows around images or objects in a document. It helps position the image with the text, allowing for better layout, readability, and a more attractive presentation of content within the document. 1.Page Wrap keeps the image fixed on the page with text wrapping on one side. 2.Optimal Page Wrap also fixes the image, but allows text to wrap on both sides, giving a more balanced and natural layout in the document.

12. What do you understand y cropping of an image?
Answer-
Cropping is the removal of unwanted outer areas from an image. Sometimes you may want to remove the unnecessary periphery of an image and put its middle part in a document. In this situation you can use the crop feature of the Writer to remove the unwanted portions.

13. How will you rotate an image inserted in Writer by any required angle?
Answer-
To rotate an image in LibreOffice Writer by any required angle: 1. Click on the inserted image to select it. 2. Right-click and choose Position and Size. 3. Go to the Rotation tab. 4. Enter the desired angle (e.g., 45°) in the Rotation Angle box. 5. Click OK to apply the rotation.

14. Write the steps to group objects that you have drawn in a Writer document.
Answer-
Steps to Group Objects in a Writer Document: 1. Select the first object by clicking on it. 2. Hold down the Shift key and click on each additional object you want to group. 3. Right-click on any selected object. 4. From the context menu, choose Group > Group.

15. How will you resize an image in a document diagonally?
Answer-
To resize an image diagonally in LibreOffice Writer: 1. Select the image by clicking on it. 2. Place the mouse pointer on any corner handle of the image. 3. The pointer will change to a diagonal double arrow. 4. Click and drag the corner handle diagonally to resize both the height and width. 5. To maintain the aspect ratio, press and hold the Shift key while dragging. 6. Release the mouse button when the desired size is reached.
