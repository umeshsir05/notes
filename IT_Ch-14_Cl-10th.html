III. Short Answer Questions

  1. How will you insert a new blank sheet?
    Sheet menu → Insert Sheet → Choose position → OK.
    Or, right-click existing sheet tab → Insert Sheet → OK.
  2. How to insert sheets from another spreadsheet?
    Insert → Sheet from File → Select file → Choose sheets → OK.
  3. How to rename a sheet?
    Right-click sheet tab → Rename Sheet → Type new name → OK.
  4. What does track changes feature do?
    Records, views, and manages edits, deletions, and formatting updates in a spreadsheet.
  5. What are macros?
    Recorded sequence of commands/keystrokes to automate tasks.
  6. How to record and run a macro?
    Record: Tools → Macros → Record Macro → Perform steps → Stop & save.
    Run: Tools → Macros → Run Macro → Select macro → Run.
  7. How to enable/disable track changes?
    Edit → Track Changes → Record → Click to enable/disable.
  8. How to link to a registered data source?
    View → Data Sources → Data Source Explorer → Select source → Drag fields into spreadsheet.
  9. What does ‘arguments in a Calc macro are passed as values’ mean?
    A copy of the data is passed, not the original cell reference.
  10. How to change a document from unshared to shared and vice versa?
    Tools → Share Spreadsheet → Check/uncheck Share option → OK.
