III. Short Answer Questions:

1.How will you insert a new blank sheet into a Calc document?
Answer -
Go to Sheet menu → Click Insert Sheet → Choose the position and click OK.
Or, right-click on an existing sheet tab → Select Insert Sheet → Click OK.

2.How will you insert sheets from another spreadsheet into your Calc file?
Answer -
Go to Insert menu → Click Sheet from File → Select the spreadsheet file → Choose the required sheets → Click OK.

3.How will you rename a sheet? Write the complete steps.
Answer -
Right-click on the sheet tab → Select Rename Sheet → Type the new name → Click OK.

4.What does the track changes feature help you to do in Calc?
Answer -
It helps to record, view, and manage all changes made in a spreadsheet, including edits, deletions, and formatting updates.

5.What do you understand by macros?
Answer -
A macro is a recorded sequence of commands or keystrokes used to automate repetitive tasks in Calc.

6.How will you record and run a macro?
Answer -
To record:
1. Go to Tools → Macros → Record Macro.
2. Perform the steps you want to record.
3. Click Stop Recording and save the macro.
To run:
4. Go to Tools → Macros → Run Macro.
5. Select the macro and click Run.

7.How can you enable and disable the track changes feature in a Calc spreadsheet?
Answer -
Go to Edit → Track Changes → Record → Click to enable or disable tracking.

8.How can you link your spreadsheet to a registered data source?
Answer -
Go to View → Data Sources → Click Data Source Explorer → Select the data source → Drag required fields into the spreadsheet.

9.What do you understand by the statement ‘arguments in a Calc macro are passed as values’?
Answer -
It means when you pass an argument to a macro, a copy of the data is passed instead of the original cell reference.

10.How can you change the status of a document from unshared to shared and vice versa?
Answer -
Go to Tools → Share Spreadsheet → Check or uncheck the Share this spreadsheet with other users option → Click OK.
