IV. Short Answer Questions.

1.What is a spreadsheet?
Answer-
A spreadsheet is an electronic worksheet used to store, organize, and analyze data in tabular form consisting of rows and columns.

2. What are the basic features of a Spreadsheet software?
Answer-
• Easy data entry and editing
• Mathematical calculations using formulas
• Functions for complex operations
• Data sorting and filtering
• Graphs and charts creation
• Formatting options for better presentation

3. What is a worksheet?
Answer-
A worksheet is a single sheet in a spreadsheet file that contains rows and columns where data is entered and managed.

4. Define the following terms with respect to Calc:
Answer-
a. Cell: The intersection of a row and a column.
b. Active cell: The currently selected cell ready to receive data.
c. Cell range: A group of adjacent cells selected together.
d. Cell address: The unique reference of a cell, such as A1 or B5.

5. What is the difference between a Spreadsheet and a Worksheet?
Answer-
A Spreadsheet is the entire file that can contain multiple worksheets. A Worksheet is a single page within the spreadsheet.

6. How columns and rows are referred in a worksheet?
Answer-
Columns are labeled with letters (A, B, C, ...) and rows are labeled with numbers (1, 2, 3, ...).

7. What will be the cell address of a cell in 5th row of 9th column?
Answer-
I5

8. How can we make a cell active?
Answer-
By clicking on it with the mouse or using arrow keys on the keyboard.

9. What is a Function?
Answer-
A function is a predefined formula that performs specific calculations automatically.

10. What is the use of Name box in the formula toolbar?
Answer-
The Name Box displays the address of the active cell and can also be used to select or name a range.

11. What is the Sheet tab?
Answer-
Sheet tabs at the bottom of the window are used to switch between different worksheets in a spreadsheet file.

12. What is the use of Input line box on the Formula toolbar?
Answer-
It displays the contents or formula of the active cell and allows editing.

13. What command options you need to open up the Format cells dialog box?
Answer-
Format → Cells or use Ctrl + 1 shortcut.

14. Which shortcut keys are used to move to first cell of the worksheet, i.e., cell A1?
Answer-
Ctrl + Home

15. Which shortcut keys are used to move to last cell of a data range in a worksheet?
Answer-
Ctrl + End

16. What is the Fill handle? What is it used for?
Answer-
The Fill Handle is a small square at the bottom-right corner of the active cell used to copy data or continue a series automatically.

17. What will be the cell address of a cell in 8th row of 19th column?
S8

18. How will you select non-adjacent cells in a worksheet?
Hold down the Ctrl key and click each cell you want to select.

19. Write the shortcut key /method for the following operations :
a. To undo the last action.
b. To paste the data in a Cell.
c. To edit the content of cell.
d. To select a column
e. To clear all the content of a row.
Answer-
a. Undo last action → Ctrl + Z
b. Paste data in a cell → Ctrl + V
c. Edit contents of a cell → F2
d. Select a column → Click on the column header or press Ctrl + Spacebar
e. Clear all contents of a row → Select the row → Press Delete key

20. What will be the cell address of a cell in 18th row of 19th column?
Answer-
S18

21. How will you fill a range of cells with 2's table using the Fill handle?
Answer-
Enter 2 in first cell, 4 in next, select both, then drag the Fill Handle downward to fill the series (2, 4, 6, 8…).

22. Differentiate between Save and Save As options of the File menu.
Answer-
• Save: Updates the existing file with current changes.
• Save As: Creates a new copy of the file with a different name or location.

23. Gauri forgot to add 2 decimal places to the numbers she has entered in her worksheet. Now she wants to insert the missed out decimal places. Write steps for her to do it.
Answer-
Steps to insert 2 decimal places:
1. Select the required cells.
2. Go to Format → Cells → Numbers tab.
3. Set Decimal places = 2.
4. Click OK.

24. Describe the procedure for the following in Calc :
a. Summing up values in a cell range.
b. Selecting continuous cell.
c. To move the contents of a cell to another cell.
Answer-
Procedure in Calc:
a. Summing up values: Use =SUM(A1:A10) or click the AutoSum (Σ) button.
b. Selecting continuous cells: Click and drag the mouse or use Shift + Arrow keys.
c. Move contents: Cut (Ctrl + X) → Select new cell → Paste (Ctrl + V).

25. What is a formula?
Answer-
Any expression that begins with an equal to (=) sign is treated as a formula. A formula is a combination of values( number), cell references and mathematical operators into an algebraic expression.
(e.g., =A1+B1).

26. What is the use of Function Wizard button?
Answer-
This button when clicked opens the function Wizard which allows us to search for various function available in Calc.
27. What is the use of the Accept button on the Formula toolbar?
Answer-
It confirms and applies the formula or data entered in the cell.

28. How can you quickly copy a formula to a range of cells?
Answer-
Use the Fill Handle to drag the formula across the desired cells.

29. Where is the Autosum button located in Calc? What is its use?
Answer-
Autosum button located on the Standard toolbar; it automatically adds up a range of cells using the SUM() function.

30. Why cell references are used in formulae instead of direct values?
Answer-
Cell references are used in formulae Because they allow automatic updates when cell values change.

31. How does Calc evaluate a complex mathematical expression?
Answer-
It follows the BODMAS rule (Bracket, Order, Division, Multiplication, Addition, Subtraction).

32. What is a Relative cell reference?
Answer-
Relative referencing is the normal way of addressing the cell like A2, B2, C4 and so on. It allows Calc to change the cell addresses in a formula when it is copied to some other rows are column.

33. What is an Absolute cell reference?
Answer-
Absolute cell reference doesn’t allows Calc to change the cell addresses in a formula when it is copied to some other rows are column.For example., $A$1).

34. How will you find number of numerical entries in a worksheet?
Answer-
In LibreOffice Calc, you can find the number of numerical entries in a worksheet using the COUNT function.
✅ Formula:
=COUNT(A1:Z100)
