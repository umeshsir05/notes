IV. Short Answer Questions.
1. What is a Word processor? Name some popular word processors.
Answer –
A word processor is a software application used to create, edit, format, and print any type of document like a report, letter, notice, circular, schedule, poster, resume, memo, or agenda etc.
Examples- LibreOffice writer, OpenOffice writer, Microsoft Word, Google Docs, word perfect, Apple iWorks pages, wordpad and Ab Word etc.
---
2. What is the use of Find and Replace feature in Writer?
Answer –
The Find and Replace feature allows us to find a occurrence of a particular word or phrase in a document and replace it with some other word or phrase.
---
3. What is the use of Ruler in the Writer window?
Answer –
The ruler is found below the formatting toolbar. You can use the ruler to change the format of your document quickly. If the ruler is not visible just click on the View->Ruler ->Ruler options from the menu bar.
---
4. List some of the features of LibreOffice Writer.
Answer –
Features of the LibreOffice writer –
1. Word processing for creating documents.
2. Styles and formatting tools.
3. Built-in templates.
4. AutoCorrect and AutoComplete.
5. Spell and grammar check.
6. Tables and columns support.
7. Insert images, charts, objects.
8. Page layout (headers, footers).
9. Mail merge feature.
10. Export to PDF/EPUB formats.
---

5. What is the function of UNDO command? How is it different from the Redo command?
Answer –
The UNDO command reverses the last action, helping to correct mistakes. The REDO command restores an action that was undone. Simply, UNDO cancels recent changes, while REDO reapplies them. Shortcuts: UNDO – Ctrl+Z, REDO – Ctrl+Y.
---
6. How will you move a line of text from one position to another in a document?
Answer –
To move a line of text from one position to another in a document:
1. Select the text you want to move.
2. Click on the Edit menu → Cut, or press Ctrl + X. The text will disappear from its place.
3. Bring the cursor to the required location.
4. Click on Edit → Paste, or press Ctrl + V.
---
7. How will you select a table in a Writer document?
Answer –
The steps to select a table are as follows:
1. Click in any cell of the table that you want to select.
2. Click on Table → Select → Table options from the menu bar.
---
8. What are the shortcut keys for copying and pasting text in a document?
Answer –
The shortcut keys for copying and pasting text in a document are:
Copy → Ctrl + C
Paste → Ctrl + V
---
9. Write the ways to select a word, a sentence, a paragraph and whole document.
Answer –
1. To select a single word, quickly double-click on that word.
2. To select a sentence , place your cursor at the start of the line, and press Shift + (↓) down arrow key.
3. To select a paragraph, place your cursor at the start of the paragraph and press Ctrl + Shift + (↓) down arrow key. You can select a sentence by using the triple click of the mouse.
---
10. How will you add an image in a document? Give a brief answer.
Answer –
To add an image in a document, place the cursor where you want the image, go to the Insert menu → Media → Gallery, browse categories, select the desired image, then right-click and choose Insert. The image will be inserted at the cursor’s position.
---
11. What do you understand by Formatting?
Answer -
Formatting means changing the look and arrangement of the text to make it more appealing and readable. Character formatting, Paragraph formatting and Page formatting are three main types of formattings in writer.

There are 3 types of formatting in Writer –

13. How does Writer indicate spelling and grammar errors in a document?
Answer -
Spelling errors are shown with a red wavy underline and grammar errors with a blue wavy underline.

14. What is the use of Thesaurus tool? Explain with an example.
Answer –
The Thesaurus tools provides access to an extensive list of alternative (both Antonyms and synonyms) words and phrases. Example: Synonyms of “Happy” → Glad, Joyful, Cheerful.
---
15. What is a bulleted list? How will you create a bulleted list in LibreOffice Writer?
Answer-
A bullet list is a small symbol that precedes every item in a bulleted list.
To create: Select text → Click on Bullets icon in Formatting toolbar.
---
16. What are Margins? How many types of margins can be defined in a Writer document?
Answer -
A margin is the space between the text and edge of a document. By default, a new document’s margin are set to Normal which means it has a one inch space between the text and each edge. There are four types of margins can be defined in Writer. These are Top, Bottom, Left, Right.
---
17. What is meant by Alignment? Name the different types of alignments.
Answer - Alignment refers to positioning text between page margins. Types: Left, Right, Center, Justify.
---
18. What is the use of the Print Preview feature?
Answer –
Print preview is quite a useful feature of writer. It let us see how the document will print once we send the file to the printer. Latest check the effect of the page layout settings.
---
19. What is the use of Header and Footer text?
Answer –
Headers and footers are identifiers that appear at the top and bottom of a document respectively. You can specify the title for the document, the author’s name, page numbers and date in the header and footer section.
---
20. What is a table? Write any two ways to insert a table in a document.
Answer –
A table is a grid of cells arranged in rows and columns. Tables are used in document to organize text, numbers, or graphical information in a concise and organised manner.
Two ways to insert:
1. Insert → Table → Select rows & columns.
2. Use the Table icon from the Standard toolbar.
