III. Short Answer Questions.

1. What is a template? Name the two main ways to create templates in LibreOffice Writer?
Answer-
A template is a file that serves as a starting point for creating a new documents. When you open a template, it is free pre-formatted in some way. For example you might use template in a word processing program that is formatted as a business letter. The two main ways to create templates in LibreOffice Writer are –
I. By saving a document as a template.
II. By using a wizard.

2. How will you create a new templates category in the Writer?
Answer-
To create a new template category follow the steps-
1.Open the library office writer and then open the template developed.
2.Click the settings icon on the templates dialogue box.
3.Click on new category option from the settings drop down menu.
4.In the dialog box that pops up, type a name for the new template category and click ok.

3. What options does a right click on any template listed in the Templates dialog box show?
Answer-
Right-clicking on a template in the Templates dialog box in LibreOffice Writer shows several options: Open, Edit, Set as Default, Rename, and Delete. These options allow users to manage templates efficiently, such as editing, organizing, or setting a default template for new documents, ensuring consistency and saving time in formatting.

4. Why type of settings does a template usually contain?
Answer-
- A template usually contains settings like:
1.Page layout (size, margins, orientation)
2.Font styles and sizes.
3.Paragraph and heading styles.
4.Header and footer.
5.Background and borders.
6.Default language and formatting.
7.Table and list styles.

5. How will you move a template from one category to another category?
Answer-
To move a template from one category to another:
1. Go to File > Templates > Manage Templates.
2. Right-click on the template you want to move.
3. Click on Move from the context menu.
4. Select the desired category from the list.
5. Click OK.

6. Distinguish between a template and a template category.
Answer-
A Template is a file that serves as a starting point for creating a new documents. When you open a template, it is free pre-formatted in some way. A template category is a folder or group that organizes templates based on purpose, like Business or Education.

7. What is Table of contents?
Answer-
A table of contents is just like the list of chapters at the beginning of a book. It list each chapter (and even section) in the document and the page number where that chapter (or sections) begins.

8. How does the Writer build a table of contents for a document?
Answer-
- LibreOffice Writer builds a Table of Contents by using headings applied to text (like Heading 1, Heading 2, etc.). It scans the document for these headings and automatically generates a ToC based on their levels and order, using the Insert > Table of Contents and Index option.
Steps:
1. Apply heading styles (e.g., Heading 1, Heading 2) to section titles.
2. Place the cursor where you want the ToC.
3. Go to Insert > Table of Contents and Index > Table of Contents, Index or Bibliography.
4. Customize settings if needed and click OK.

9. Write any two advantages of using templates for creating documents.
Answer-
Two advantages of using templates for creating documents:
1. Saves Time: Templates provide a ready-made structure, so you don’t have to format documents from scratch every time.
2. Ensures Consistency: Templates maintain uniform formatting and style across all documents, ensuring a professional and consistent look.

10. What are the two ways in which a table of contents can be deleted from a document?
Answer-
(a) Using Right-Click Menu: 1.Right-click on the Table of Contents.
2.Select Delete Index from the context menu.
(b) Using Navigator:
1.Open Navigator (press F5).
2.Right-click on the Table of Contents entry under Indexes.
3.Choose Delete.

11. How will you update a table of contents?
Answer-
Steps to Update the ToC:
1. Click anywhere in the Table of Contents.
2. Right-click, and from the context menu, choose Update Index.
Writer will now update the ToC to reflect the latest changes in the document.

12. What is the use of the Type tab of the Table of Contents, Index or Bibliography dialog box?
Answer-
The "Type" tab in LibreOffice Writer is used to define the basic settings for creating a Table of Contents, Index, or Bibliography. It lets you choose the type, set a title, and decide how entries are collected from the document. This tab is essential for structuring and generating these elements accurately within your document.
