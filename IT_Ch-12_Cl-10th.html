II. Fill in the blanks

  1. A Template is a document that we use to create other documents.
  2. All documents in Writer are created using one template or another.
  3. You can create templates in two ways: by saving a document as a template or by using a wizard.
  4. To edit a template you need to open the Templates dialog box by clicking File → Templates → Manage template.
  5. Templates contain Formatting specifications and other settings that can be modified and saved within a template.
  6. The letter wizard allows creating templates for business and personal letters.

III. Short Answer Questions

  1. What is a template? Name two main ways to create templates in LibreOffice Writer.
    Answer: A template is a file that serves as a starting point for creating new documents. Two main ways:
    1. Save a document as a template.
    2. Use a wizard.
  2. How will you create a new templates category in Writer?
    Answer:
    1. Open LibreOffice Writer and then the template library.
    2. Click the settings icon in the Templates dialog box.
    3. Select New Category.
    4. Type a name and click OK.
  3. What options does a right-click on any template show?
    Answer: Open, Edit, Set as Default, Rename, Delete.
  4. What type of settings does a template usually contain?
    Answer:
    • Page layout (size, margins, orientation)
    • Font styles and sizes
    • Paragraph and heading styles
    • Header and footer
    • Background and borders
    • Default language and formatting
    • Table and list styles
  5. How will you move a template from one category to another?
    Answer:
    1. Go to File > Templates > Manage Templates.
    2. Right-click the template to move.
    3. Select Move.
    4. Choose desired category and click OK.
  6. Distinguish between a template and a template category.
    Answer: A template is a pre-formatted file for new documents. A template category is a folder organizing templates by purpose.
  7. What is Table of Contents?
    Answer: A Table of Contents lists chapters or sections of a document with page numbers.
  8. How does Writer build a Table of Contents?
    Answer: Writer uses headings applied to text (Heading 1, 2, etc.), then Insert > Table of Contents & Index generates it automatically.
  9. Two advantages of using templates:
    Answer:
    1. Saves time: ready-made structure.
    2. Ensures consistency: uniform formatting and style.
  10. Two ways to delete a Table of Contents:
    Answer:
    1. Right-click → Delete Index
    2. Navigator (F5) → Right-click entry → Delete
  11. How will you update a Table of Contents?
    Answer: Click in ToC → Right-click → Update Index.
  12. Use of Type tab in ToC dialog box?
    Answer: Defines basic settings, title, entry collection method, and structure for ToC, Index, or Bibliography.
