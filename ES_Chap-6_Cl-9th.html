IV. Short Answer Questions

1. What do you mean by hardware and software?
Answer-
Hardware: hardware are the physical parts of a computer system. We can touch, hold or carry the hardware parts from one place to another place. Keyboard, mouse, monitor printer, CPU speakers, etc are some example of computer hardware. Software: software is a collection of computer programs, procedures and documentation that performs different task on a computer system. Software are the set of instructions or computer programs that guide the computer hardware to perform different types of task. Example -, MS Windows, Linux, MS Word, Excel, PowerPoint, Photoshop and CorelDRAW etc .
---

2. What are input devices? Give examples of input devices.
Answer-
Input devices are the devices that help us to enter data and instructions into the computer. Keyboard, mouse, scanner, web camera, microphone, joystick, light pen, trackball, MICR ocr, OMR, BCR, ETC. are some example of input devices.
---
3. What are output devices? Give examples of output devices.
Answer -
Output devices are the devices that are used to display or communicate the output or results of the data processing done by the computer. Some example of output devices are monitor, printer, speaker, headphones, plotter etc .
---
4. What are storage devices? Give examples of storage devices.
Answer-
devices that are used to store data permanently are called storage devices. Examples of storage devices are hard disc, floppy disc, CD, DVD, Blu ray disc, pen drive, etc.
---
5. What are the two main types of computer memory?
Answer-
The two main types of computer memory are-
1.Primary memory (RAM, ROM), which is fast and volatile, used for temporarily storing actively accessed data.
2.Secondary memory (hard drives, SSDs), which is slower but permanent, storing files and programs long-term and retaining data even when power is off.
---
6. What is the use of RAM and ROM?
Answer-
RAM (Random Access Memory): It is the primary memory that stores data and instructions temporarily while the computer is working.
ROM (Read Only Memory): It is permanent memory that stores start-up instructions and cannot be modified easily.
