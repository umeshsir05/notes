III. Short Answer Questions.

1.Differentiate between a worksheet and a workbook (or spreadsheet).
Answer -
• Worksheet: A sheet (or Worksheet) is a grid of numerous rectangular boxes arranged in horizontal rows and vertical columns.
• Workbook: A workbook in calc is also referred to as a spreadsheet. It is a file that we open in LibreOffice calc to do our work.

2.What is data sorting in Calc? How will you arrange data in an ascending order in the worksheet?
Answer –
Sorting data means arranging it in a particular order. Arranging data in a worksheet in a particular order (like a ascending order or descending order) allows us to find desired data entries quickly.
• Steps for Ascending Order:
1. Select the data range.
2. Click Sort Ascending button on the Standard toolbar.

3.What is AutoFilter? What is its utility?
Answer -
• AutoFilter: A feature that allows you to filter and display only the rows that meet certain criteria.
• Utility: Quickly hides unwanted data and shows only relevant information.

4. What is data analysis in the context of a spreadsheet program?
Answer –
Data Analysis as a process of Inspecting, cleaning, transforming, and modeling data with the aim of deriving useful information from it, which can further be used in an organization’s decision making process.

5. What is data consolidation? Can it be implemented on multiple sheets?
Answer -
Data consolidate is a feature that allows us to consolidate or put together data from various locations within a spreadsheet or from multiple spreadsheet, at one place.
Yes, it can be implemented on multiple sheets using Data → Consolidate.

6. Differentiate between the Goal Seek and Solver features.
Answer -
• Goal Seek: Finds the required input value to achieve a specific goal in one cell.
• Solver: A more advanced tool that adjusts multiple variables to achieve the desired result.

7. What is meant by What-If analysis? List three types of What-If analysis tools available in Calc.
Answer –
The “what if” analysis is a powerful feature in calc and a many other spreadsheet programs. During ‘what if’ analysis we change the values in cells to see how this changes will affect the outcome of a formula on the worksheet.
• Types of What if analysis are : Goal Seek, Scenario, Data Table.

8. What is the scenario tool? Discuss in brief.
Answer -
A Scenario is a set of values that calc saves for a group of cells. Whenever we run a scenario Calc automatically substitutes the saved cell values into a connected formula to give us an output.

9. How will you open the Navigator to work with scenarios?
Answer -
To open navigator, click on the view menu and from the drop down list select the navigator option.

10. How do we track values in a scenario?
Answer –
To learn which values in the scenario affect other values choose Tool → Detective → Trace Dependents.

11. What do you understand by data filtering? How many types of filters are there in Calc?
Answer –
When a Sheet contains a large amount of data, it can be difficult to find information quickly. In that case filter can be used to narrow down the data in your worksheet allowing you to view only the information you need.
• Types of filters -
1. Standard Filter
2. AutoFilter
3. Advanced Filter

12. What does the Calc’s Subtotal feature help us to do?
Answer –
The Subtotal feature creates a summarized outline of data in a worksheet. It allows us to divide data into groups so that it can be easily viewed and analyzed.
