III. Short Answer Questions

  1. Differentiate between a worksheet and a workbook:
    • Worksheet: A grid of cells in rows and columns.
    • Workbook: A Calc file (spreadsheet) containing one or more worksheets.
  2. What is data sorting in Calc?

    Sorting arranges data in ascending or descending order for quick lookup.

    Steps:
    1. Select the data range.
    2. Click Sort Ascending button on the Standard toolbar.
  3. What is AutoFilter?
    • Feature: Filter and display rows meeting criteria.
    • Utility: Quickly hide irrelevant data.
  4. What is data analysis?

    Inspecting, cleaning, transforming, and modeling data to derive useful information for decision making.

  5. What is data consolidation?

    Combining data from various locations or multiple sheets in one place. Implementable via Data → Consolidate.

  6. Differentiate Goal Seek and Solver:
    • Goal Seek: Finds required input for a single cell goal.
    • Solver: Adjusts multiple variables for desired result.
  7. What is What-If analysis?

    Changing cell values to see how it affects formula outcomes. Types: Goal Seek, Scenario, Data Table.

  8. What is Scenario tool?

    Stores sets of values that Calc automatically substitutes to give output.

  9. How to open Navigator for scenarios?

    View → Navigator from menu.

  10. How to track values in a scenario?

    Tools → Detective → Trace Dependents.
