III. Short Answer Questions:

1.Define each of the following:
  a. RDBMS  b. Table  c. Report  d. Query
Answer-
a. RDBMS -
RDBMS (Relational Database Management System ) is a type of database management system that allows us to create and work with the relational database.
b.Table -
A table is a group of related records. Tables are frequently classified by the application for which they are primarily used.
c. Report- Report is a database object that allows us to present data from a table or query table in a printable format.
d. Query- Queries are database objects that we create to extract selected data from the database tables.

2.What are the advantages of using a Database Management System?
Answer-
Advantages of a Database are -
1.Facilitate Sharing of Data: Multiple users can use the same centrally stored data simultaneously for different purposes.
2.Prevent Duplication of Data: Centralized storage minimizes repeated or duplicate data entries.
3.Ensure Data Consistency: No data duplication; all users see the same updated data.
4.Make Data Validation Possible: Tools ensure data correctness and reduce errors.
5.Ensure Data Security: Only authorized users can access data using authentication.
6.Ensure Confidentiality: Access is limited to authorized users as per privacy rules.

3.State the relationship between the various components of the Table - Data items, records, and fields.
Answer-
Relationship between Data Items, Fields, and Records:
Data Item:
It is the smallest unit of data, such as a single value (e.g., a student’s name or roll number).
Field:
A field is a column in a table that contains a specific type of data for all records (e.g., Name, Age, Class).
Record:
A record is a complete set of related fields that represents one entry in the table (e.g., details of one student).
Relationship:
Many data items form a field, and several fields together form a record in a database table.

4.Which are the different methods you can use create a table in LibreOffice Base?
Answer-
Different methods to create a table in LibreOffice Base:
1.Create Table in Design View:
You can manually define field names, data types, and properties. Allows full control over the table structure.
2.Create Table using Wizard:
Provides step-by-step guidance to select fields from sample tables and set primary keys easily.
3.Create Table in SQL View:
You can write SQL commands (like CREATE TABLE) to design the table manually using SQL syntax.

5.What is a primary key? Explain with the help of a suitable example.
Answer-
A primary key is a field or column in a database table that identifies each record in the table uniquely. Example:
In a Student table:
Roll_No Name Class
101 Ravi 10
102 Sita 10

Here, Roll_No is the Primary Key because each student has a unique roll number.

6.What is a query? State any one advantage of using a query in a database.
Answer-
Queries are database objects that we create to extract selected data from the database tables. The design view of the LibreOffice Base allows us to criteria for extracting selective data from database tables in the form of query tables.

Advantage:
A query allows users to quickly find and view specific data from large databases, saving time and effort.

7.What is a form? List any two of its advantages.
Answer-
A form is an important component of a database. It is always linked to the Table (or query) object. It provides an easy interface to the user to modify and insert regards. Any changes that a user makes to the data in a form also get reflected related table or query table.
Advantages:
1. Makes data entry easier with text boxes, dropdowns, and buttons.
2. Helps reduce errors by providing a controlled way to input data.

8.What is a report? List different ways of creating a report in LibreOffice Base.
Answer-
Report is a database object that allows us to present data from a table or query table in a printable format.
Different ways of creating a report in LibreOffice Base:
1. Using Report Wizard:
Provides step-by-step guidance to create a report quickly.
2. Using Report Builder (Design View):
Allows users to manually design and customize the report layout.
9.What is a relationship in context of the LibreOffice Base tables?
Answer-
When we link one table to the other on the basis of the Primary key, it is called setting a relationship. We can set a relationship between two or more tables.
10.How many types of relationships we can establish between the Base tables?
Answer-
In LibreOffice Base, we can establish three types of relationships between tables.
1. One-to-One Relationship:
Each record in one table is linked to only one record in another table. Example - In a school database each student has only one student ID and each student ID is assigned to only one person.
2. One-to-Many Relationship:
A single record in one table can be related to multiple records in another table. Example - Each customer can have many sales orders.
3. Many-to-Many Relationship:
Records in one table can relate to multiple records in another table, and vice versa. Example - Customers can purchase various products and products can be purchased by many customers.
