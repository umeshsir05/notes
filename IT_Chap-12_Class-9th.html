V. Short Answer Questions.

1. Discuss the various types of keys available on a computer keyboard.
Answer –
There are various keys available on a keyboard:
1. Function Keys (F1–F12): Perform special functions in software.
2. Punctuation Keys: Include symbols like comma, period, brackets, etc.
3. Alt Key: Used with other keys for shortcuts (e.g., Alt + F4).
4. Arrow Keys: Move the cursor in four directions.
5. Shift Keys: Used for capital letters and special symbols.
6. Ctrl Key: Used with other keys for control commands.
7. Delete Key: Deletes characters or items to the right.
8. Backspace Key: Deletes characters to the left.
9. Caps Lock: Toggles capital/small letters.
10. Enter, Esc, Tab, PgUp/PgDn, Home/End, Print Screen, Spacebar: Help in navigation, screen capture, and spacing.

2. Differentiate between Alt key and the Shift key.
Answer –
Alt key is a modifier key used with other keys, like Alt + F4 to close programs. Shift key helps type capital letters and upper symbols like @, #, *. There are two Shift keys on the keyboard. Both keys are used in combination with others to perform special functions.

3. What do you understand by Guide Keys? Name the Guide keys of a
a. Computer keyboard
b. Numeric keypad
Answer –
Guide keys are special keys on the keyboard that help touch-typists place their fingers correctly on the home row without looking. These keys have a small bump that can be felt by fingers.
a. On the computer keyboard:
• F key (for the left hand)
• J key (for the right hand)
b. On the numeric keypad:
• 5 key (has a small bump for finger placement)

4. Explain the role of typing ergonomics in practicing typing.
Answer –
Typing ergonomics ensures correct posture, hand placement, and keyboard positioning while typing. It helps improve comfort, speed, and accuracy. Practicing proper ergonomics prevents Repetitive Stress Injuries (RSIs) and reduces bodily strain. It allows users to type efficiently for longer periods without discomfort, supporting both health and productivity at the workplace.

5. Mention the finger allocation of keys of the Bottom Row of computer keyboard.
Answer –
The bottom row (Z, X, C, V, B, N, M, ,, ., /) uses the following finger allocations:
• Left Little Finger: Z
• Left Ring Finger: X
• Left Middle Finger: C
• Left Index Finger: V, B
• Right Index Finger: N, M
• Right Middle Finger: , (comma)
• Right Ring Finger: . (full stop)
• Right Little Finger: / (slash) and Right Shift key
• Thumbs: Spacebar (usually right thumb)

6. What is meant by Home keys? What is their importance in typing?
Answer –
The home keys are the keys on the computer keyboard, our fingers rest on when not typing. On the standard QWERTY keyboard, the home keys for our left hand are A, S, D, F and the home keys for our right hand are J, K, I, ; (semicolon). For both hands, the thumbs rest on the spacebar.
